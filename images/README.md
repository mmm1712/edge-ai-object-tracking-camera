# Project Images

This folder contains the main photos, diagrams, and demo files used in the project README.

## Files

- `tracking-demo.gif` - demo of the camera following the detected object
- `original-camera.jpg` — the first ESP32-CAM version that depended on a laptop
- `upgraded-camera.jpg` — the upgraded camera using Grove Vision AI V2 and XIAO ESP32-S3
- `hardware.jpg` — the main hardware used to build the upgraded version
- `system-diagram.png` — diagram showing how object detection and servo control work
- `cad-assembly.png` — OpenSCAD render of the custom pan/tilt camera structure
